# JavaScript DOM Manipulation

This project contains a series of JavaScript exercises focused on DOM manipulation, event handling, and API requests.

## Learning Objectives

- How to select HTML elements in JavaScript
- Differences between ID, class and tag name selectors
- How to modify an HTML element style
- How to get and update an HTML element content
- How to modify the DOM
- How to make requests with XmlHTTPRequest and Fetch API
- How to listen/bind to DOM events and user events

## Tasks Description

### 0. Color Me
Update the text color of the header element to red using `document.querySelector`.

### 1. Click and turn red
Update the text color of the header to red when clicking on the element with id `red_header`.

### 2. Add `.red` class
Add the class `red` to the header element when clicking on the element with id `red_header`.

### 3. Toggle classes
Toggle between `red` and `green` classes on the header element when clicking on `toggle_header`.

### 4. List of elements
Add a new `<li>Item</li>` element to the list when clicking on `add_item`.

### 5. Change the text
Update the header text to "New Header!!!" when clicking on `update_header`.

### 6. Star wars character
Fetch and display a Star Wars character name using the Fetch API.

### 7. Star Wars movies
Fetch and list all Star Wars movie titles using the Fetch API.

### 8. Say Hello!
Fetch and display a "hello" translation from an API.
